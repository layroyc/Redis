（1） 查看key是否存在：
set knames wy
 exists knames 

（2） 查找满足 x开头的   的keys（需要在数据库中添加若干个 x开头的 几个单词）：
scan 0 MATCH x* count 14
 keys x*

（3） 查看key的超时时间：setex 设置过期时间，存的时候就给定一个过期时间

ttl code
 

（4） 遍历key：

 keys *

（5） 返回key的值的序列化：
dump name
 

 

2、 string类型数据的命令操作：

（1） 设置键值：

（2） 读取键值：

（4） 数值类型自减1：

strlen person
