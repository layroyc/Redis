    k-v结构   单个key 最大能存储512MB，约等于  多少 个文字呢？？自己算 哼
增(改)：  set key value   例子： set adminName 'zhangsan'
查：  get key         例子： get adminName
查询所有的key：   keys *  例子：keys *
删除key:  del key     例子： del adminName       删除返回的是数量
String 的 特有命令：
批量增： mset key1 value key2 value
批量查： mget key1 key2
字符串截取：  getrange key start end   【字符串的索引到结束】
先查再存： getset   格式  getset key value     将给定的key值设置value，并返回key的旧值。  应用场景：工资！！！
(重要)过期时间：setex 设置过期时间，存的时候就给定一个过期时间。验证码就用它   setex code 60 898
先时间，后是value
字符串长度： strlen key 的长度， 不存在的key 的长度是0
(重要) 自增（减） incr  场景： 抖音点赞，产品的热销排行   自增的值一定是数字
定量增： incrby 场景：一次加多个物品      incry mycode 6
(重要) 自减： decr key
定量减： decrby 场景：一次减多个物品    decrby mycode 9   游戏、

字符串的追加： append key value 追加的位置是 value的末尾

增： hset key

例子：存一个 hset person name “wy”

三、list --双端队列结构   先进后出，排序是按照 插入的顺序排序的   （有序的，可以重复）
例： 沙和尚  猪八戒 孙悟空  唐僧 王源（插入的） 刘备 关羽 张飞 赵子龙
左边增： lpush key xxx
右边增： rpush key xxx
遍历： lrange key 0 -1
左边删除： lpop key
右边删除： rpop key
插入： linsert key before "刘备" "王源"    例子：linsert hero before 'liubei' 'roy'
查单：lindex key 0   例子：lindex hero -2/5
删除指定的值  比较复杂  lrem key count value      例子：lrem hero count value
count > 0 表示 从头向尾巴   从左到右 搜索，移除何 value 相等的元素，数量为count
count < 0 表示 从尾巴向头   从右到左 搜索，移除何 value 相等的元素，数量为count 的绝对值
count = 0 表示 移除表中所有与value相等的值
注意：因为是list 的数据类型， 所以 值 是可以相同的，比如 有2个 roy,2个 lisi

四、set集合   （无序的，不能重复）
增加：sadd key value
查条数：scard  获取key的成员数量
查全：smembers key 返回集合中的所有成员value
按索引查单？？？ 有没有   sindex names 2 没有的，，，，，没有顺序
判断元素是否在集合中    sismember key value
指定删除：srem key value
随机删除！！！ 用途很广    spop key    例子：spop names
作业：用java代码写，把班级33个人的名字，形成一个集合，点一个人名，
就把他移除，再次点名是点32人中的随机一个

随机查： srandmember key

五：zset 或 sortset    有序的set集合。可以有序当是不重复
他是set的扩展板，每个元素都关联了个 double 类型的分数，根据分数的大小，进行排序。
排序的规则是 从小到大，分数可以并列即 分数可以重复，成员（value） 必须唯一
底层是哈希表实现的 添加删除 很快
增加： zadd key 分数 值,pnames
遍历：zrange key 0 -1  遍历值
        zrange key 0 -1 withscores 加上withscores 带分数，不带 直接全部显示值   遍历值和下标
查条数：zcard key
指定的删除：移除集合中的一个或多个成员（value） zrem key value

以上是常用命令
